# Operating Systems: Overview

### Types of Operating Systems
* **Batch OS** - Groups similar jobs for sequential processing.
* **Time-Sharing OS** - Switches CPU time rapidly between multiple users.
* **Real-Time OS** - Built for time-critical systems.
    * **Hard RTOS:** Guarantees strict deadlines.
    * **Soft RTOS:** Prioritises tasks but allows minor delays.
* **Distributed OS** - Manages independent nodes as a single system.
* **Network OS** - Manages server-side resources and security.
* **Mobile OS** - Optimized for handheld devices and power efficiency.

### Core OS Algorithms
* **FCFS (First-Come, First-Served):** Executes processes in the order of arrival.
* **SJF (Shortest Job First):** Prioritizes the task with the smallest execution time.
* **Round Robin (RR):** Assigns a fixed time slice to each process in a cycle.
* **Priority Scheduling:** Executes tasks based on assigned importance levels.
* **MLFQ (Multilevel Feedback Queue):** Dynamically adjusts process priority based on history.
